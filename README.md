# 莫璃云社-新一代智能生态的社区APP

## 介绍
莫璃云社APP是为动漫/影视/游戏/软件/插件使用者们提供解决方案的一站式资源分享交流社区。我们添加了多个及详细的分类目录与标签。通过APP，您可以方便且快捷的找到所需要的资源并下载使用.
## 软件特色
- 软件Ui全面重构美化、功能升级、bug修复.
- 完整的圈子模块、圈主权限体系
- 用户个性化系统：用户背景、性别、创作者、企业认证等功能
- 完善的推送：私信消息、系统通知、帖子消息
- 逐步完善的智能生态计划，开启智能社区生态新体验
